1. Exploracin inicial

El dataset Iris contiene mediciones de 150 flores de 3 especies diferentes:

  • Setosa
  • Versicolor
  • Virginica

Cada flor tiene 4 mediciones (en cm):

  • Longitud de spalo
  • Ancho del spalo
  • Longitud del ptalo
  • Ancho del ptalo

# Histogramas de las 4 caractersticas
fig, axes = plt.subplots(2,2,figsize=(12,10))
fig.suptitle('Distribucin de Caractersticas por Especie',fontsize=16)

features = iris.feature_names

for idx, feature in enumerate(features):
    ax = axes[idx//2,idx%2]

    for species in df['species_name'].unique():
        data = df[df['species_name']==species][feature]
        ax.hist(data, alpha=0.6, label=species, bins=15)

    ax.set_xlabel(feature, fontsize=10)
    ax.set_ylabel('Frecuencia', fontsize=10)
    ax.legend()
    ax.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

Observaciones:

  1. Separabilidad de especies:
    • La especie Setosa es claramente disinguible de las otras dos
    • Versicolor y Virginica tienen cierta superposicin
  2. Caractersticas ms distintivas:
    • Longitud y ancho del ptalo son los mejores discriminadores
    • Setosa tiene ptalos significativamente ms pequeos
  3. Correlaciones:
    • Fuerte correlacin positiva entre lontitud de ptalo y longitud de spalo (0.87)
    • Fuerte correlacin entre longitud y ancho del ptalo (0.96)
  4. Distribuciones:
    • Todas las caractersticas siguen distribuciones aproximdamente normales
    • Algunas especies muestran bimodalidad en ciertas caractersticas

Aplicaciones potenciales:

Este anlisis demuestra que con slo 4 medidas simples podramos construir un modelo de Machine Learning para clasificar especies de Iris con alta precisin.
